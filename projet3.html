Projet Personnel • Bot Discord + API ChatGPT

Bot Discord propulsé par ChatGPT

Bot Discord en Python interagissant avec les utilisateurs et générant des réponses contextuelles via l'API ChatGPT.

Contexte

Projet personnel réalisé en autonomie pour explorer l’utilisation d’API externes et renforcer mes compétences en Python. Objectif : développer un bot Discord capable d’interagir avec les utilisateurs d’un serveur, en s’appuyant sur l’API de ChatGPT pour générer des réponses dynamiques et contextuelles.

Conçu de manière simple et légère : un unique fichier Python, un fichier .env pour la gestion sécurisée des clés d’API, et une intégration directe avec Discord.

Python • Discord • API ChatGPT

Projet personnel • Intégration API • .env sécurisé

Ma participation

Principales missions :

  • Développement du bot en Python
  • Connexion à l’API Discord via une bibliothèque dédiée
  • Intégration de l’API ChatGPT pour générer des réponses
  • Gestion des messages utilisateurs et réponses automatiques
  • Tests du bot directement sur un serveur Discord

Le bot répond de manière fluide grâce aux réponses dynamiques générées par l’IA.

Bilan

Découverte concrète de l’intégration d’une API d’IA dans une application réelle. Renforcement de ma maîtrise de Python et meilleure compréhension de la sécurité des clés d’API et des échanges entre services.

Résultats

Bot Discord opérationnel : réponses automatiques via ChatGPT, gestion sécurisée des clés, déploiement et tests réalisés sur un serveur Discord personnel.
